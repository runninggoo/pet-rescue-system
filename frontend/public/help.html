欢迎使用宠物救助收养系统

本系统覆盖从"救助发现 → 领养审核 → 健康管理 → 后续跟踪"全业务流程，为流浪宠物找到温暖的家。

系统简介 角色说明 领养流程 志愿者流程 常见问题
🏠 帮助中心
基于 Spring Boot + Vue.js 的前后端分离 Web 应用，覆盖上海地区流浪宠物救助收养的完整数字化管理流程。
🐾
宠物管理
支持多维度筛选（品种/紧急程度/健康状态），快速找到心仪宠物
📋
领养审核
多申请人横向对比，规范审核流程，支持补充材料与协议签署
🏥
健康档案
疫苗、体检、治疗全记录，支持到期提醒，守护宠物健康
❤️
志愿者体系
积分成长、每日签到、排行榜，激励志愿者持续参与公益
📊
数据统计
实时掌握领养成功率、区域分布、季节性趋势，辅助科学决策
🔗
回访跟踪
领养后定期回访，关注宠物适应情况，记录回访历史
👥 角色说明
系统支持五大角色，各司其职、协同工作
👑
系统管理员
拥有系统最高权限，管理用户账户、宠物信息、机构数据、全部统计数据。
用户管理 数据统计
🏢
机构管理员
管理本机构宠物、审核领养申请、录入健康档案、发布志愿任务。
审核申请 健康档案
🏠
领养人
浏览宠物、提交领养申请、补充材料、签署领养协议、完成领养流程。
申请领养 签署协议
🌟
志愿者
接取并完成任务、获得成长积分、每日签到、查看个人排行榜与等级。
接任务 获积分
🏥
宠物医院
查看宠物健康档案、录入疫苗和体检记录、管理医疗资源信息。
疫苗记录 体检记录
💡 领养流程
从发现宠物到完成领养的完整业务流程
🐾 领养申请 → 审核 → 签约 → 回访
1
浏览宠物
在宠物列表页筛选品种、紧急程度，找到心仪宠物
→
2
提交申请
填写养宠经验、居住条件、家庭成员等信息
→
3
等待审核
管理员/机构审核，可要求补充材料或直接拒绝
→
4
签署协议
审核通过后签署领养协议，正式成为领养人
→
5
领养成功
宠物正式交接入住新家，机构可进行回访跟踪
🌟 志愿者成长流程
完成任务获得积分，提升等级，解锁更多公益荣誉
🌟 接单 → 完成任务 → 获得积分 → 升级
1
浏览任务
在任务管理页查看救助站发布的各类志愿任务
→
2
接取任务
点击接单，任务状态变为"进行中"，按时完成
→
3
获得积分
任务完成后获得积分奖励，每日签到也有额外积分
→
4
等级提升
累计积分达到阈值自动升级，共6个等级：新手上路→公益之星
→
5
榜单荣誉
每周积分排行榜 TOP10，积分历史可查，等级徽章展示
