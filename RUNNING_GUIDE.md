## 🔧 常见问题解决

### 1. 数据库连接失败
**问题**：`java.sql.SQLException: Access denied for user 'root'@'localhost'`
**解决**：
- 检查MySQL用户名和密码
- 修改 `application.yml` 中的数据库配置
- 确保MySQL服务已启动

### 2. 编译错误
**问题**：找不到符号或类错误
**解决**：
- 确保所有Maven依赖已下载
- 清理项目缓存：`mvn clean`
- 重新编译项目

### 3. 前端页面加载慢
**问题**：页面加载时间长
**解决**：
- 使用本地Web服务器而不是直接打开文件
- 检查网络连接
- 清除浏览器缓存

### 4. API调用失败
**问题**：`404` 或 `500` 错误
**解决**：
- 确保后端服务已启动
- 检查API地址是否正确
- 查看后端控制台错误日志
