### 第三步：前端运行

**重要说明**：前端必须通过 Vite 开发服务器运行（而不是直接打开 HTML 文件），否则 API 请求无法代理到后端。

1. **安装依赖**（首次运行或依赖变更后）：
   ```bash
   cd C:/Users/33169/Desktop/计算机/毕业设计/项目/frontend
   npm install
   ```

2. **启动前端开发服务器**：
   ```bash
   npm run dev
   ```
   访问 http://localhost:5173/ （会自动跳转到登录页面）

3. **旧方式（不推荐）**：直接双击 `public/login.html` — **API 请求会失败**，因为没有开发服务器代理请求到后端。

## 🔧 常见问题解决

### 1. 数据库连接失败
**问题**：`java.sql.SQLException: Access denied for user 'root'@'localhost'`
**解决**：
- 检查MySQL用户名和密码
- 修改 `application.yml` 中的数据库配置
- 确保MySQL服务已启动

### 2. 编译错误
**问题**：找不到符号或类错误
**解决**：
- 确保所有Maven依赖已下载
- 清理项目缓存：`mvn clean`
- 重新编译项目

### 3. 前端页面加载慢
**问题**：页面加载时间长
**解决**：
- 使用本地Web服务器而不是直接打开文件
- 检查网络连接
- 清除浏览器缓存

### 4. API调用失败
**问题**：`404` 或 `500` 错误
**解决**：
- 确保后端服务已启动
- 检查API地址是否正确
- 查看后端控制台错误日志

### 领养申请流程（6状态）
```
提交申请(待审核0) → 审核通过(已通过1 → 自动转待签署4) / 拒绝(2) / 待补充(3重新提交)
                  ↓
              签署协议(待签署4) → 完成(5已完成)
```
- 审核通过后同一宠物的其他申请自动被拒绝（防抢先机制）
- 签署协议需要领养人本人操作
