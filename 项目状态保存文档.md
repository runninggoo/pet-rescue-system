### v1.2.3（2026-04-20）UI收尾完善 + Bug修复

#### UI收尾完善

- **模块C**：pet-detail.html 补充渐变装饰背景

- **模块E**：新建 index.html 入口页（导航栏+Hero+6张功能卡片+4步领养流程+CTA+页脚）

#### Bug修复：pet-list.html el-dialog Vue隔离问题

**问题现象**：管理员登录后，点击「新增宠物」或「删除宠物」按钮，弹出弹窗后页面变暗遮罩，**点击弹窗内任何按钮均无响应**。

**根因**：`el-dialog` 将内容挂载到 `document.body`，创建独立Vue实例，与根 `#app` 实例隔离，所有 `@click` 回调无法访问根Vue的data/methods。

**修复方案**：两处 `el-dialog` → 原生 div 覆盖层（`.cat-overlay` + `.cat-panel` 样式复用）。

#### Bug修复：pet-favorite 收藏唯一键冲突

**问题现象**：收藏→取消→再次收藏时报错：`Duplicate entry '1-20' for key 'uk_user_pet'`

**根因**：数据库唯一键 `uk_user_pet (user_id, pet_id)` 与软删除字段 `deleted` 是独立的两套机制——软删除不等于释放唯一键。

**修复方案**：使用 MySQL `INSERT ... ON DUPLICATE KEY UPDATE`，在数据库层面原子处理冲突：
```xml
<insert id="insertOrReactivate">
    INSERT INTO pet_favorite (user_id, pet_id, deleted, created_at)
    VALUES (#{userId}, #{petId}, 0, NOW())
    ON DUPLICATE KEY UPDATE deleted = 0, created_at = NOW()
</insert>
```
