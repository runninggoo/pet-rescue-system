## 🐛 已知问题和解决方案

### 1. 中文字段名问题
**问题**: Shelter实体中使用了中文字段名 `入驻状态`
**解决方案**: 已改为英文字段名 `entryStatus`
**影响文件**: Shelter.java, ShelterServiceImpl.java, ShelterMapper.java

### 2. 构造器注入问题
**问题**: 多个Service和Controller使用了@Autowired字段注入
**解决方案**: 已改为构造器注入，更符合Spring最佳实践
**影响文件**: 所有Service和Controller类

### 3. 重复方法问题
**问题**: AdoptionApplicationServiceImpl中有重复的findByCondition方法
**解决方案**: 已删除重复方法，保留正确实现
**影响文件**: AdoptionApplicationServiceImpl.java

### 4. MyBatis映射问题

## 📝 关键注意事项

### 1. 数据库配置
- 需要确保MySQL 8.0.42已启动
- 检查utf8mb4编码设置
- 验证上海地区救助所测试数据

### 2. Spring Security配置
- JWT密钥配置
- 密码加密配置
- 角色权限配置

### 3. 智能匹配算法
- Haversine公式距离计算
- 权重配置：距离40% + 容量30% + 医疗等级30%
- 默认推荐3个救助所

### 4. 权限控制
- admin: 查看所有数据
- institution_admin: 查看本区域数据
- adopter: 只能查看待领养宠物
- volunteer: 查看已入驻救助所

## 📚 项目关键信息

### 技术栈
- Spring Boot 2.7.0
- MyBatis Plus 3.5.3.1
- JWT + Spring Security
- MySQL 8.0.42
- Vue.js (前端)

### 核心功能
- 上海地区救助所管理
- 智能匹配算法
- 健康记录管理
- 任务协调系统
- 微信小程序

### 数据库设计
- shelter表: 救助所信息
- pet表: 宠物信息
- user表: 用户信息
- adoption_application表: 领养申请
- follow_up_record表: 回访记录
